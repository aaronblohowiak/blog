title: Skype In Ubuntu Karmic Core Dump date: 2009/08/07 23:49:37 -0700 tags: [] author: Aaron Blohowiak alias: /skype-in-ubuntu-karmic-core-dump If you are running ubuntu karmic and you `sudo aptitude upgrade`, and find you cannot start skype after reboot, then you may need to remove your existing install, set up the www.medibuntu.com repository and then aptitude install skype-static-oss. Then, things should work again. Hopefully, the issue with the main distribution of skype will be resolved shortly.
